# GitAgent 🚀

GitAgent is a modern web application built to simplify Git workflows and improve developer productivity.

## Features ✨
- Clean and modern UI
- Fast development workflow
- Version control integration
- Scalable project structure

## Tech Stack 🛠️
- React.js
- Node.js
- Vite
- JavaScript
- CSS

## Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/Bhavi6115/GitAgent.git
cd GitAgent
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the development server
```bash
npm run dev
```

### 4. Open in browser
Vite will generate a local URL like:

```bash
http://localhost:5173
```

Open that in your browser.

## Build for Production

```bash
npm run build
```
